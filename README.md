About Me
I am an undergraduate student majoring in Geoinformatics at the University of Seoul.
My current focus is on my graduation thesis, which investigates the spatial characteristics of river flow response to precipitation — with particular emphasis on Curve Number (CN) and Lag-Time analysis using GIS and statistical methods.
Alongside my research, I am developing practical software projects including a remote desktop tool and a game built in Unity.
